|       食物名称 | 重量 | 花费 | 单价 |
| -------------: | ---- | ---- | ---- |

|       新鲜大蒜 |      | 1.6  | 5    |
|           皮蛋 |      | 1    | 1    |
|           橘子 |      | 2.4  | 2    |

早上把国庆之前买的一个土豆、一个芋头、2个鸡蛋在预定的时间（电饭锅功能）内蒸好了，不然又得是一个很忙的日子呀，加上现在上班打卡很难打上，也是很尴尬啊！土豆蒸着吃也可，和番薯类似，但是没有那么甜。

晚上买完东西回来就把橘子和皮蛋消灭了，哈哈哈哈哈哈哈哈哈哈。

把大蒜切段；西红柿切一个，切丝；一些瘦肉切丝；昨天买的海带浸泡后切丝。准备完成之后，切了一些肥肉末到米中，然后剩下的肉切块与大蒜、蒜头、姜片拌匀，加入盐、酱油搅拌，一会儿之后拿去电饭锅中按煮饭键炖【生活告诉我，肉还是应该炖久一点，不然感觉像生的一样】。

炖好肉和饭之后，烧滚水，然后依次放入海带丝、肉、西红柿、剩下的一点平菇，在快煮好之后，加入炖肉的汤汁当油，然后继续煮几分钟后装入碗中。吃了之后，这个汤味道真不错，有点胡辣汤的味儿了，哈哈哈哈哈，改良版的胡辣汤。

今日吃了 1/4 的饭，喝了所有的汤，吃了几块炖的肉，嗯，有点撑。不过现在感觉不知道是那碗菜放咸了，现在嘴里感觉好咸呐！以后应当注意这件事，不然是越来越胖了。

今日剩余一个西红柿、白菜，还有 3/4 的米饭，炖的肉还有一大半。

### 10 月 11 日

| 食物名称 | 重量 | 花费 | 单价 |
| -------: | ---- | ---- | ---- |
| 冷冻鸡腿 |      | 8    | 8.5  |
|     生菜 |      | 3.4  | 4    |

冷冻鸡腿不可多吃，若食用肉类，应早起晨市买鱼，此比较实惠。

生菜焯水过后美甚，也许是第一次吃的缘故吧！

红薯蒸熟过后十分甘甜，趁热吃较好，冷了风味没那么好。

今日 2 餐均为捞面条，中午 2 鸡腿，1 颗生菜，3 两红薯（后又把剩下的红薯吃掉了），1 些面条。整体来说在吃完肉之后会有点撑，下次注意一点，不要吃的太撑，对身体不好。同时应该注意自己能够吃饱（不至于吃撑）的食量，这样也不至于太过浪费。

今日煮菜的顺序是：烧滚水（同时把蒸的鸡腿的汁液导入煮沸，下入生菜，捞出后下入面条。这种煮法不会导致最后的汤汁被面条吸走。

### 10 月 9 日

| 食物名称 | 重量 | 花费 | 单价 |
| -------: | ---- | ---- | ---- |
|     冬瓜 |      | 1.9  | 2    |
|     豆芽 |      | 1    | 1.5  |
|     芋头 |      | 2.7  | 3.5  |

豆芽与冬瓜在水烧至沸腾后放入，再次沸腾后放入盐、油煮之，熟后捞出。下入面条，再次煮沸后放入十三香、黄豆酱，然后备一空碗盛之。将里面面条捞出放入之前盛豆芽与冬瓜的碗中，最后浇入一些剩下的汁，`吃之甚美` 。后续如若可以，可做冬瓜汤来喝，_`注：此时配点肉就很完美了，可晚上买菜肉的质量不太好，而早上又起不太来，太难受了`_，想必也是十分甜美的吧！

这些时日，每次买蔬菜都会剩下很多，下次需要注意。且晚上不宜吃太饱，应节约便宜行事。此次买菜剩余豆芽大半，芋头没动。

### 10 月 8 日

|           食物名称 | 重量 | 花费 | 单价 |
| -----------------: | ---- | ---- | ---- |
|               萝卜 |      | 2.2  | 2.5  |
|     白菜（上海青） |      | 1.4  | 3    |
| 草鱼（非鱼头鱼肚） |      | 8    | 6    |

1. 西红柿煮酱：水`少量`，烧至沸腾后，倒入西红柿，煮开，加`油`，`盐`，然后煮到汤色浓稠，出锅。`味道还可`
2. 白菜、茄子如上法煮制，`吃之尚可`
3. 白萝卜，若存放久而切之，可见白环雪花状，此已糠，食之营养减；`白萝卜若不煮久，则食后有辛辣味`，久之难祛。新鲜之白萝卜，煮之无辛辣味，可见存久味道难吃，而与常价相差较大者，应是存久之货，此应少买，不然也是亏损且食之无味，浪费哉。
4. 新鲜白萝卜，煮之，在水滚烫后放入十三香，食之尚可；而晚饭加入黄豆酱加热后拌之，`亦有一番风味`，_`「注：十三香买月余，已然结团，此前期煮菜之失误，导致后期食材皆未放此物」`_，
5. 鸡腿泡水后直接清蒸，别有一番味道
6. 鱼肉加盐，油拌后，加入红薯淀粉，放入盆中搅拌，蒸煮之后，`亦有可赞之处`，比前些时候乱煮乱炖乱放调料好及。
